Google Form (recommended — no Cloud Console)

Works for anyone — no developer console, no OAuth, no email whitelist.

Use the JobTrail template form — click "Make a copy"

  1. Click the template link above — it'll prompt "Make a copy," which drops a private copy into your own Google Drive, independent of the original
  2. In your copy, click Responses → the green Sheets icon, to create your own linked spreadsheet
  3. Click the ⋮ menu → Get pre-filled link (some accounts show this as "Pre-fill form" instead — same feature)
  4. Type the exact placeholder text below into each question, then Get link → Copy link
Job ID (linking key)JC_JOBID
TitleJC_TITLE
CompanyJC_COMPANY
LocationJC_LOCATION
SalaryJC_SALARY
SourceJC_SOURCE
URLJC_URL
Fit Score (optional)JC_FITSCORE
Matching Skills (Paragraph, optional)JC_MATCHING
Missing Skills (Paragraph, optional)JC_MISSING
Status (optional)JC_STATUS

Minimum needed: Job ID + Title + URL. All other questions are optional — missing ones are silently skipped. Job ID is the key to VLOOKUP between both sheets.

Don't want to rely on a template you don't control? Build your own form instead
  1. Go to forms.google.com and create a blank form
  2. Add 6 "Short answer" questions, in any order: Title, Company, Location, Salary, Source, URL
  3. Important: leave every question not required (the toggle in the bottom-right of each question should be off) — a required-but-empty field can silently block a submission
  4. Click the Responses tab, then the green Sheets icon, to create a linked spreadsheet
  5. Click the ⋮ menu → Get pre-filled link, then continue with the placeholder table above

Analysis Form (2nd form — resume suggestions + cover letter)

This form captures the deep analysis. Linked to its own sheet, VLOOKUP on Job ID to connect with the Jobs sheet. Auto-submits whenever you do a full per-job score.

Build the analysis form from scratch instead
  1. Go to forms.google.com, create a blank form
  2. Add 3 questions: Job ID (Short answer), Resume Suggestions (Paragraph), Cover Letter (Paragraph)
  3. Leave all not required
  4. Link a spreadsheet via Responses → Sheets icon
